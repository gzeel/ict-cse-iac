# Windows

## VSCode

Installeer de Remote-SSH extensie. Je kunt dan verbinding maken met de ontwikkel VM die je gemaakt hebt.
Je krijgt dan ook de bestanden op de VM te zien in de File Explorer van VSCode.

Er zijn diverse andere extensies die je kunt installeren zoals thema's en dingen die je code beter leesbaar maken zoals Prettier.

## Terminal

Je kunt de default Windows Terminal gebruiken maar bv ook iets als Putty, Wezterm, Alacritty etc.
